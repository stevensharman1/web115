FCC Full Stack JavaScript Outline

This table summarizes the JavaScript Algorithms and Data Structures portion of the freeCodeCamp Full Stack curriculum (917 steps total). It demonstrates the use of thead, tbody, tfoot, rowspan, and colspan.

Section Steps Description
Code Editors 5 In these lectures, you will learn how to work with code editors and IDEs. You will learn various concepts about the most popular code editor, VS Code such as its installation, how to create a project in it, keyboard shortcuts, and extensions.
Variables and Strings 62 Start with the basics of JavaScript: how to declare variables, and how to work with text data using strings. You'll learn about concatenation, template literals, and string methods such as length, slice, toUpperCase, toLowerCase, and more.
Booleans and Numbers 37 Learn about different data types, including numbers for mathematical operations and booleans for logic and decision-making in your code. Practice comparison operators, arithmetic operations, and boolean logic.
Arrays 45 Discover how to store, access, update, and manipulate ordered collections of data. Work with array methods such as push, pop, shift, unshift, slice, splice, and learn how arrays can be nested inside other arrays.
Functions 55 Learn how to declare reusable blocks of code that can be called when needed. Practice passing arguments, returning values, using function expressions, and arrow functions to make your code modular and efficient.
Objects 48 Explore how to use objects to store key–value pairs. Learn to access properties with dot and bracket notation, update and delete properties, and use objects as flexible containers for structured data.
ES6 Features 39 Practice new syntax introduced in ECMAScript 6, including let and const, arrow functions, template literals, default parameters, rest and spread operators, and destructuring assignments to simplify your code.
Debugging 30 Learn techniques to identify and fix problems in your programs. Use console.log, breakpoints, and careful reading of error messages to correct both syntax and logical errors in your JavaScript.
Basic Algorithm Scripting 80 Begin solving small algorithmic challenges. Write step-by-step instructions to manipulate numbers, strings, and arrays, preparing you for larger projects and improving your problem-solving skills.
Intermediate Algorithm Scripting 60 Work through more advanced algorithmic problems that require combining loops, conditionals, and array methods. Strengthen your ability to break down complex challenges into smaller, solvable steps.
Regular Expressions 35 Dive into pattern matching with regular expressions. Learn special characters, flags, and quantifiers to search, extract, and replace text inside strings effectively.
Debugging with Regex 18 Apply regular expression knowledge to debug text-related issues in your programs. Practice building and refining patterns to locate errors and unexpected inputs.
Basic Data Structures 50 Learn how to build and manipulate data structures such as stacks and queues. Practice working with nested arrays and objects to store and organize information in useful ways.
Intermediate Data Structures 42 Explore more advanced structures like Maps and Sets, along with linked lists. Learn when and how to apply these structures to solve different types of problems.
Functional Programming 65 Discover the principles of functional programming. Work with higher-order functions, map, filter, and reduce to process data in a clean and declarative way. Learn about immutability and avoiding side effects.
Object Oriented Programming 57 Learn how to model real-world entities with classes and objects. Practice using constructors, inheritance, encapsulation, and methods to create reusable and scalable code.
Intermediate Algorithm Practice 40 Strengthen your skills with a new set of practice problems that combine arrays, objects, loops, functions, and string manipulation. These prepare you for the final certification projects.
Project 1–5 80 Apply everything you have learned by building five real-world projects: Palindrome Checker, Roman Numeral Converter, Caesars Cipher, Telephone Number Validator, and Cash Register. Completing these earns the certification.
